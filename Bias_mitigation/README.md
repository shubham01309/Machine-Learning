# Bias Mitigation in Machine Learning #

## Project Overview
This repository contains the code and data for a machine learning project focused on mitigating bias in machine learning models. The main objective is to address and reduce bias in datasets and models through various techniques. This project includes steps for training models, evaluating performance, and explaining model predictions.

### Models
This project explores several machine learning models to mitigate bias, including:

Decision Tree: A simple, interpretable model used as a baseline.

Explainable Boosting Machine (EBM): An advanced model that provides interpretable predictions.
